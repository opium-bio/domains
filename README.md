# Domains 🌐

## Overview

High-performance domain control service built with Golang and Redis.

## Features

- Fast domain registration
- Redis-powered storage

## Prerequisites

- Go
- Redis
- Mongodb

## Installation

```bash
git clone https://github.com/opium-bio/domains.git 
cd domains
go build -o domains
```
