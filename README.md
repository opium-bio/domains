# Domains 🌐

## Overview

High-performance domain control service built with Rust and Redis.

## Features

- Fast domain registration
- Redis-powered storage

## Prerequisites

- Rust
- Redis
- Cargo

## Installation

```bash
git clone https://github.com/opium-bio/domains.git 
cd domains
cargo build --release
```

## Development

```bash
# Run tests
cargo test

# Build
cargo build

# Run
cargo run
```
